Politique de confidentialité

Albexia accorde une grande importance à la protection de votre vie privée. Cette page vous explique quelles données nous collectons, pourquoi, et comment nous les utilisons.

1. Données collectées

Albexia collecte un minimum de données personnelles, uniquement lorsque vous effectuez une action volontaire :

  • Inscription à la newsletter : votre adresse email est transmise à Formspree (formspree.io), service tiers de gestion de formulaires, pour vous envoyer la newsletter hebdomadaire. Nous ne partageons cette adresse avec aucun autre tiers.
  • Soumission d'un outil : les informations que vous renseignez (nom de l'outil, URL, description, email optionnel) sont reçues par email via Formspree et utilisées uniquement pour examiner votre soumission.
  • Favoris : vos outils favoris sont stockés localement sur votre appareil via localStorage. Ces données ne quittent jamais votre appareil et ne sont pas accessibles par nos serveurs.

2. Finalité des données

  • Envoi de la newsletter hebdomadaire (email)
  • Traitement des soumissions d'outils (email + informations soumises)
  • Amélioration du service (pas de tracking individuel)

3. Base légale du traitement

Le traitement de vos données est basé sur votre consentement explicite : vous nous fournissez vos données volontairement en remplissant un formulaire. Vous pouvez retirer ce consentement à tout moment.

4. Conservation des données

Vos données d'inscription à la newsletter sont conservées tant que vous êtes abonné. Un lien de désabonnement est inclus dans chaque email. Après désabonnement, vos données sont supprimées dans un délai de 30 jours.

5. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie des données vous concernant
  • Droit de rectification : corriger des données inexactes
  • Droit à l'effacement : demander la suppression de vos données
  • Droit d'opposition : vous opposer au traitement de vos données
  • Droit à la portabilité : recevoir vos données dans un format lisible

Pour exercer ces droits, contactez-nous à : albexia.hello@gmail.com

6. Cookies et stockage local

Albexia n'utilise aucun cookie de tracking ou publicitaire. Le seul stockage utilisé est localStorage pour sauvegarder vos favoris localement sur votre appareil. Vous pouvez effacer ces données depuis les paramètres de votre navigateur.

7. Services tiers

  • Formspree (formspree.io) : traitement des formulaires. Politique de confidentialité Formspree
  • Google Fonts : chargement des polices de caractères. Peut impliquer le transfert de votre adresse IP à Google.
  • GitHub Pages : hébergement du site. Politique de confidentialité GitHub
  • Unsplash / SoundHelix : contenus médias. Ces services peuvent collecter des données d'utilisation.

8. Sécurité

Le site utilise le protocole HTTPS pour sécuriser toutes les communications. Aucune donnée de paiement n'est collectée ou stockée sur ce site.

9. Modifications

Cette politique de confidentialité peut être mise à jour. La date de dernière modification est indiquée en haut de cette page. Nous vous encourageons à la consulter régulièrement.

10. Contact

Pour toute question relative à cette politique ou pour exercer vos droits : albexia.hello@gmail.com
