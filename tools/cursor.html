Questions fréquentes

Cursor est-il vraiment basé sur VS Code ?
Oui, Cursor est un fork de VS Code. Toutes vos extensions et configurations VS Code fonctionnent directement. La migration prend moins de 5 minutes.
Cursor vs GitHub Copilot : lequel est meilleur ?
Cursor est plus puissant pour les gros projets car il comprend l'architecture globale. Copilot est plus discret et reste dans votre éditeur actuel. Cursor est le choix des développeurs qui veulent l'IA-first.
Quels modèles IA utilise Cursor ?
Cursor donne accès à Claude Sonnet (Anthropic), GPT-4o et o1 (OpenAI) selon votre besoin. Vous choisissez le modèle pour chaque conversation.
Cursor peut-il lire tous mes fichiers ?
Oui, Cursor indexe votre projet entier. Vous pouvez aussi sélectionner des fichiers spécifiques à inclure dans le contexte pour des interactions plus précises.
