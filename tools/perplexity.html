What is Perplexity?

Perplexity is an AI-powered search engine that combines the power of large language models with real-time web access.

Unlike ChatGPT which can invent information, Perplexity always cites its sources. Each answer includes numbered references you can verify — making it a reliable tool for research.

FAQ

Is Perplexity free?
Yes, the free version is very capable with unlimited searches and real-time web access. Pro adds advanced models and file uploads.
Perplexity vs Google — what's the difference?
Perplexity synthesizes multiple sources into a direct answer. Google shows you a list of links. Perplexity is faster for research.
Does Perplexity hallucinate?
Much less than other AI tools because it cites sources. However, always verify important information from primary sources.
